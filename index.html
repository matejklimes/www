World Wide Web

Je označení pro systém prohlížení, ukládání a odkazování dokumentů nacházejících se na internetu.

Dokumenty (webové stránky) si prohlížíme pomocí webového prohlížeče, jsou uloženy na webových serverech a jsou navzájem propojeny pomocí hypertextových odkazů zapisovaných ve formě URL.

Vznik

Autorem je Tim Berners-Lee, který jej vytvořil při svém působení v CERNu.

Navrhl jazyk HTML a protokol HTTP, napsal první webový prohlížeč a spustil první webový server na světě.

V říjnu roku 1994 založil W3C, které dohlíží na další vývoj Webu.

Jakou roli plní webový server? Uveďte příklady webových serverů.

Server je bezobslužný program, který přijímá a obsluhuje požadavky klientů. Webový server je připojen k počítačové síti a přijímá požadavky. Tyto požadavky vyřizuje počítači, který požadavek vznesl, vrací odpověď.

Příklady webových serverů:

  • Apache
  • NGINX
  • Microsoft Internet Information Service
Jak se nazývá program označovaný jako webový klient? Co je jeho úkolem? Vyjmenujte nejpoužívanější webové klienty.

Klient (prohlížeč) je program, který komunikuje s uživatelem a podle pokynů se obrací na jednotlivé servery, bere z nich informace a zobrazuje je.

Například:

  • Google Chrome
  • Microsoft Edge
  • Safari
Jak probíhá komunikace mezi webovým klientem a webovým serverem?

Základem služby WWW je komunikace mezi webovým prohlížečem a serverem přes protokol HTTP/HTTPS.

Při jejich spolupráci fungují 2 mechanismy:

  • protokol HTTP, pomocí něhož komunikuje prohlížeč se serverem
  • jazyk HTML, formuje samotnou stránku, kterou zobrazuje prohlížeč
Co je HTTP?

HTTP je internetový protokol určený pro komunikaci s WWW servery. Slouží pro přenos hypertextových dokumentů.

Společně s elektronickou poštou je HTTP nejvíce používaným protokolem, který se zasloužil o obrovský rozmach internetu v posledních letech.

Proč se stále častěji využívá protokol HTTPS?

HTTPS je v informatice protokol umožňující zabezpečenou komunikaci v počítačové síti.

Využívá protokol HTTP spolu s protokolem SSL nebo TLS. Díky tomu je bezpečnější, a právě proto se využíva častěji.

Co jsou a k čemu slouží tzv. cookies? Připojte obrázek ukazující uložená cookies k některé internetové stránce.

Jako cookie (anglicky koláček, oplatka, sušenka) se v protokolu HTTP označuje malé množství dat, která WWW server pošle prohlížeči, který je uloží na počítači uživatele.

Cookies běžně slouží k rozlišování jednotlivých uživatelů, ukládají se do nich uživatelské předvolby apod..

Vypište přehledně jednotlivé fáze fungování internetových vyhledávačů.

Fungování vyhledávačů má v podstatě 3 snadné kroky:

  • crawlování (sbírání dat do databáze vyhledávače)
  • indexace (příprava indexu)
  • výdej výsledků
Vysvětlete zkratku HTML a pojem hypertext.

HTML je zkratka pro Hypertext Markup Language, což je název značkovacího jazyka používaného pro tvorbu webových stránek, které jsou propojeny hypertextovými odkazy.

Hypertext je způsob strukturování textu, který není lineární. Obsahuje tzv. hyperlinky neboli česky (hypertextové) odkazy. Nejznámějším takovým systémem je World Wide Web.
